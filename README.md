## Запуск

Установите все зависимости
```bash
npm i
```

Для запуска в режиме разработки выполните
```bash
npm run dev
```

Линт кода: проверит все JavaScript-файлы и файлы стилей
```bash
npm run lint
```

Для production сборки - сервер запуститься на `3000` порту
```bash
npm run build
npm run start
```

Запуск тестов
```bash
npm run test
```
